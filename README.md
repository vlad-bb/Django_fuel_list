# **Django Fuel List**

## [Fuel List](https://fuel-list-2022.fly.dev/)

### What is the Fuel List?

#### Fuel List is having next function:

- Shows average fuel prices at gas stations in Ukraine

- Map with nearest gas stations around user (in progres)

- Fuel calculator (in progres)
